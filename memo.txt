留意事項
1.キーボードのeventの番号を調べる。
キーボードとマウスのeventの番号については逐一確認する必要がある。再起動してるとよく変わってる
logicoolのキーボードはeventの後ろにledsが付いている方を使う。何故かそっち
event番号の調べ方は「cat /proc/bus/input/devices」
実際に動いていることを確認するときは「hexdump /dev/input/event*(イベント番号)」を用いることとする。

マウス操作について
マウスカーソルを垂直(上)に持って行く時
event.code = REL_Y event.value =  -1 * spd event.type = EV_REL
マウスカーソルを垂直(下)に持って行く時
event.code = REL_Y event.value =  1 * spd event.type = EV_REL
マウスカーソルを水平(左)に持って行く時
event.code = REL_X event.value = -1 * spd event.type = EV_REL
マウスカーソルを水平(右)に持って行く時
event.code = REL_X event.value = 1 * spd event.type = EV_REL
マウスの左ボタンを押した時
event.code = BTN_LEFT event.value = 1 event.type = EV_KEY
マウスの左ボタンを離した時
event.code = BTN_LEFT event.value = 0 event.type = EV_KEY
マウスの右ボタンを押した時
event.code = BTN_RIGHT event.value = 1 event.type = EV_KEY
マウスの左ボタンを離した時
event.code = BTN_RIGHT event.value = 0 event.type = EV_KEY
マウスのホイールを下げた時
event.code = REL_WHEEL event.value = -1 event.type = EV_REL
マウスのホイールを上げた時
event.code = REL_WHEEL event.value = 1 event.type = EV_REL
マウスの真ん中ボタンを押した時
event.code = BTN_MIDDLE event.value = 1 event.type = EV_KEY
マウスの真ん中ボタンを離した時
event.code = BTN_MIDDLE event.value = 0 event.type = EV_KEY
